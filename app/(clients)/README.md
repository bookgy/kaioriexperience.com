# Clients

Esta carpeta usa un `route group` de Next.js.

`app/(clients)` sirve solo para agrupar páginas de clientes y no aparece en la URL.

Ejemplos:

- `app/(clients)/cliente-a/page.js` -> `/cliente-a`
- `app/(clients)/hotel-demo/page.js` -> `/hotel-demo`

Estructura sugerida:

```text
app/
  (clients)/
    nombre-cliente/
      page.js
```
