# Landings

Esta carpeta usa un `route group` de Next.js.

`app/(landings)` sirve solo para agrupar landings y no aparece en la URL.

Ejemplos:

- `app/(landings)/mi-landing/page.js` -> `/mi-landing`
- `app/(landings)/oferta-especial/page.js` -> `/oferta-especial`

Estructura sugerida para nuevas landings:

```text
app/
  (landings)/
    nombre-landing/
      page.js
```
