# Docs internas

Esta carpeta es solo para documentacion interna del proyecto.

No forma parte del sitio publico porque esta fuera de `app/` y fuera de `public/`.

Reglas:

- Guardar aqui premisas, criterios, decisiones y notas internas del proyecto.
- No crear rutas web a partir de estos archivos.
- No mover esta documentacion a `public/`.
- Si algun dia existiera una URL `/docs`, debe mantenerse bloqueada para robots.

Archivos iniciales:

- `SEO.md`
- `premisas-proyecto.md`
